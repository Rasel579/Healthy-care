# Healthy-care
This android app using for looking to yourself, health like heart beat, pulse etc
 - Added all information about your health (pulse, heart beat, pressure)
 
 # Stack
 - Clean Architecture
 - MVVM
 - FireBase Database
 - Koin
 - Coroutines
 - Room
 - Pagination
